JavaIO章节

IO流的分类

可以按照以下几种方式分类

按数据单位进行分类：字节流（8bit），字符流（16bit）

按照数据的流向进行分类：输入流，输出流

按照流的角色进行分类：节点流，处理流

IO流的体系划分

节点流（文件流）

FileReader

FileWriter

FileOutputStream

FileInputStream

缓冲流

BufferedInputStream

BufferedOutputStream

BufferedReader

BufferedWriter

缓冲流的使用要套间在相应的节点流的上面，它的内部内置了一个缓冲区，缓冲区的大小是8kb。

向流中写入字节时，不会直接写到文件，先写到缓冲区中直到缓冲区写满， BufferedOutputStream才会把缓冲区中的数据一次性写到文件里。使用方法 flush()可以强制将缓冲区的内容全部写入输出流

使用flush方法可以将缓冲区的内容手动的清空

在管理流的时候，因为缓冲流是包在相应的节点流的上面的，所以只需要关闭最外层的那个流，内层的流会自动的经进行关闭。

转换流

InputStreamReader：将一个字节的输入流转换成字符的输入流

OutputStreamWriter：将一个字符的输出流转换成字节的输出流

转换流的主要作用是可以做转码操作

标准输入流输出流

System.in 和 System.out 分别代表了系统标准的输入设备和输出设备，默认的输入设备是键盘，输出设备是显示器

System.in的类型是InputStream，System.out的类型是PrintStream

Class PrintStream 是OutputStream的子类，也是FileOutputStream的子类

打印流

打印流中实现了基本数据类型的数据格式转化为字符串输出

PrintStream 和 PrintWrite 的输出不会抛出IO异常、具有自动刷新功能

PrintStream 和 PrintWrite

数据流

DataInputStream 和 DataOutputStream

作用：用于读取或者写出基本数据类型的变量或者字符串

对象流

ObjectInputStream 和 OjbectOutputSteam

用于存储和读取基本数据类型数据或对象的处理流。它的强大之处就是可 以把Java中的对象写入到数据源中，也能把对象从数据源中还原回来

序列化：用ObjectOutputStream类保存基本类型数据或对象的机制

反序列化：用ObjectInputStream类读取基本类型数据或对象的机制

ObjectOutputStream和ObjectInputStream不能序列化static和transient修饰的成员变量

对象的序列化机制：对象的序列化，允许把内存中的Java对象转换成平台无关的二进制流，从而允许把这种二进制流持久的保存在起来，

当其他程序获取这种二进制的流，就可以恢复成原来的的java对象

如果想要实现对象的序列化则需要实现Serializable接口，并且重新指定serialVersionUID，

案例3、通过流的读写操作完成文件的复制操作

    //图片的复制操作
    @Test
    public void testImg() throws IOException {
        File fileSrc = new File("mm.jpg");
        File fileDesc = new File("mm1.jpg");

        FileInputStream fi = new FileInputStream(fileSrc);
        FileOutputStream fo = new FileOutputStream(fileDesc);

        byte[] cbuf = new byte[512];

        while((len = fi.read(cbuf))!=-1){
            //写操作

        fo.close();
    }

案例4、通过缓冲流完成文件的复制操作

    //使用缓冲流进行文件的复制
    @Test
    public void testBufferCopy() throws IOException {
        File fileSrc = new File("mm.jpg");
        File fileDest = new File("mm2.jpg");

        FileInputStream fi = new FileInputStream(fileSrc);
        FileOutputStream fo = new FileOutputStream(fileDest);

        BufferedInputStream bis = new BufferedInputStream(fi);
        BufferedOutputStream bos = new BufferedOutputStream(fo);

        byte[] cbuf = new byte[10];
        int len=0;
        while((len = bis.read(cbuf))!=-1){
            bos.write(cbuf,0,len);
        }

        bis.close();
        bos.close();
    }

案例6、通过装欢流对文件的编码集进行一个修改

    //使用转换流对问价的编码进行一个修改
    @Test
    public void u8ShifGBk() throws IOException {
        File file1 = new File("tt.txt");
        File file2 = new File("tt_bgk.txt");

        //创建节点流
        FileInputStream fis = new FileInputStream(file1);
        FileOutputStream fos = new FileOutputStream(file2);

        //创建转换流
        InputStreamReader isr = new InputStreamReader(fis,"utf-8");
        OutputStreamWriter osw = new OutputStreamWriter(fos,"gbk");

        char[] cbuf = new char[20];
        int len = 0;

            osw.write(cbuf,0,len);
        }
        isr.close();

    }

案例7、输入流，从键盘输入字符串，要求将读取到的整行字符串转成大写输出。然后继续 进行输入操作，直至当输入“e”或者“exit”时，退出程序

    public static void main(String[] args) throws IOException {
        InputStreamReader isr = new InputStreamReader(System.in);
        BufferedReader br = new BufferedReader(isr);

        while(true){
            System.out.println("请输入字符串：");
            String data = br.readLine();

            if ("e".equalsIgnoreCase(data) || "exit".equalsIgnoreCase(data)){
                System.out.println("程序结束");
                break;
            }
            String upperCase = data.toUpperCase();
            System.out.println(upperCase);
        }
        br.close();
    }

案例8、数据流的读写操作

    //数据流的写操作
    @Test
    public void testDataIo() throws IOException {
        DataOutputStream dos = new DataOutputStream(new FileOutputStream(new File("data.txt")));
        dos.writeUTF("牛一从");
        dos.flush();
        dos.writeInt(22);
        dos.flush();
        dos.writeBoolean(true);
        dos.flush();
        dos.close();
    }
    //数据流的读操作
    //进行读操作的时候要和写入时的数据的顺序一致
    @Test
    public void readDataTest() throws IOException {
        DataInputStream dis = new DataInputStream(new FileInputStream(new File("data.txt")));

        String name = dis.readUTF();
        int age = dis.readInt();
        boolean isMale = dis.readBoolean();

        System.out.println("name = " + name);
        System.out.println("age = " + age);
        System.out.println("isMale = " + isMale);
        dis.close();
    }
