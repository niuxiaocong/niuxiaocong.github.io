迭代器

可以进行迭代的内容，Array、Arguments、Set、Map、String、TypedArray、NodeList

迭代器的工作原理

​ 创建一个指针对象，指向当前数据结构的起始位置

​ 第一次调用对象的next方法，指针自动指向数据结构的第一个成员

​ 接下来不断的调用next方法，指针不停的向后面移动，一直移动到最后一个成员

​ 每一次调用next方法返回一个包含value和done属性的对象

        const xiyou = ['金角大王','银角大王','铁角大王'];
        let iterator = xiyou[Symbol.iterator]();
        console.log(iterator.next())
        console.log(iterator.next())
        console.log(iterator.next())
        console.log(iterator.next())


        const lishi = {
            name:'终极三国',
            stus:[
                '关羽','张飞','刘备','马超','黄忠','曹操'
            ],
            [Symbol.iterator](){
                //索引变量
                let index = 0

                let _this = this

                return{
                    next:function () {
                        if (index < _this.stus.length){
                            const result = {value:_this.stus[index],done: false}
                            index++
                            return result
                        }else{
                            return {value:undefined,done: true}
                        }
                    }
                }
            }
        }
        for (let v of lishi){
            console.log(v)
        }

ES8新特性

async和await

async和await两种语法结合可以让异步代码像同步代码一样

async函数

async 函数的返回值为Promise对象

promise对象的结果由async函数执行的返回值决定

        async function fn() {
            //返回一个字符串
            //return '牛小牛'
            //返回的结果不是一个Promise类型的对象，返回的结果就是成功的Promise对象
            //return;

            //抛出错误，返回的结果是一个失败的Promise
            //throw new Error('出错了')

            //返回的结果如果是一个Promise对象
            return new Promise((resolve, reject) => {
                //resolve("成功的数据")
                reject('失败的数据')
            })

        }

        const result = fn()
        console.log(result)

await表达式

await必须卸载async函数中

await右侧的表达式一般为Promise对象

await返回的是promise成功的值

await的promise失败了，会抛出异常，并且通过try…catch进行捕获处理

例子

发送成功

        //创建Promise对象
        const p = new Promise((resolve,reject)=>{
            //成功
            resolve('成功的数据')
        })

        async function main(){
            let result = await p;
            console.log(result)
        }
        main()

发送失败

        //创建Promise对象
        const p = new Promise((resolve,reject)=>{
            //失败
            reject('失败的数据')
        })

        async function main(){
            try{
                let result = await p;
                console.log(result)
            }catch(e){
                console.log(e)
            }    
        }
        main()

async和await的结合发送AJAX请求

        //在这里借用的是Jquery的接口
        function sendAJAX(url) {
            return new Promise((resolve, reject) => {
                //创建Ajax对象
                const x = new XMLHttpRequest();

                //初始化
                x.open('GET',url);

                //发送
                x.send();

                //绑定事件
                x.onreadystatechange = function () {
                    if (x.readyState === 4){
                        if (x.status>=200 && x.status<300){
                            //成功啦
                            resolve(x.response)
                        }else{
                            //失败
                            reject(x.status)
                        }
                    }

                }
            })
        }
