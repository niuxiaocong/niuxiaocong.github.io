IO是什么

IO(input/output) 即输入输出，数据在设备之间的传输。它的应用场景有数据的输入和输出、文件操作、网络通信方面都有应用。

IO的分类

根据流的流向分成输入流和输出流，输入流通常以Inputxxx开头，输出流通常以Outputxxx开头。

根据流的操作单位不同可以分成字节流和字符流，字节流通常以xxxStream结尾，字符流通常以xxxReader或者xxxWrite结尾

IO流的操作主要在java.io包中，这个包下有4个顶级类，分别是InputStream、OutputStream、Reader、Writer

字符流进行文件的读写

Reader 读文件内容到系统中

Writer 写内容到系统文件中

InputStream 读取文件到系统中

InputStream 不适合处理文字， 因为由于GBK每个汉字两个字节，UTF-8每个汉字三个字节，如果定义的byte数组长度不能刚好读到每个汉字的最后一个字节的话，就会出现乱码。

Stream 更加适合处理图片和音频。

解决方案：

1、定义足够大的byte数组，吧文字都包含进去（适合处理小文件，不适合处理大文件）。

2、使用字符流FileReader，通过字符可以保障内阁字符不乱码。

3、使用转换流吧FileInputStream转换成FileReader。
